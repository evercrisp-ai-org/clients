# Sample Content

Place your organization's existing content here. These samples are used to generate the voice profile (Session 1 in START_HERE.md) and serve as the reference standard for voice checks on all future content.

## What to Include

- Blog posts
- Social media posts (LinkedIn, Facebook, Twitter/X, etc.)
- Email newsletters
- Podcast scripts or transcripts
- Presentation scripts
- Any written content that represents the voice you want the system to capture

## How Much

- **Minimum:** 10 pieces for a basic voice profile.
- **Recommended:** 30+ pieces for a robust voice profile with clear pattern detection.
- **More is better:** The more samples, the more accurately the system can distinguish the real voice from AI-generated approximations.

## Accepted Formats

- Markdown (`.md`) -- preferred
- Plain text (`.txt`)
- Word documents (`.docx`)
- PDFs (`.pdf`)
- URLs listed in a `sources.md` file (one URL per line with a short description)

## Organization

You can organize samples however you prefer (flat list, by channel, by date). The voice-profile generation prompt will read all files in this folder.

## Updating

Add new samples as the voice evolves. If the voice materially changes (new voice owner, major brand pivot), re-run Session 1 from START_HERE.md to regenerate the voice profile.
